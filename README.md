## Project Title
Desjardins Insurance - Information Security Policy (ISMS)

## Purpose
This repository contains the final project deliverables for an Information Security Policy aligned with ISO/IEC 27001:2022 for Desjardins Insurance.

The policy focuses on protecting confidentiality, integrity, and availability of information assets, while addressing privacy, legal, regulatory, and governance obligations in Canada.

## Repository Approach
This project is structured so each file can be committed individually, step by step.

Planned deliverables include:
- Core policy document in Markdown
- PlantUML diagrams for governance and ISMS lifecycle views
- Supporting documents (glossary, version history, compliance mapping)
